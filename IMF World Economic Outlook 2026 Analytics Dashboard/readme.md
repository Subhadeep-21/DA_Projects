- # IMF World Economic Outlook 2026 Analytics Dashboard

## Objective

Analyze IMF macroeconomic forecasts using interactive dashboards.

## Tools

Power BI | Excel | Power Query

## Key Insights

- Engineered an advanced Excel analytics dashboard covering 190+ countries and 4 key macroeconomic indicators: GDP growth, inflation, fiscal balance, and current account trends.
- Integrated data from multiple sources using ETL, cleaned and analyzed 2,700+ data points using Pivot Tables, XLOOKUP, IF formulas, conditional formatting, and structured tables for reliable reporting.
- Designed 6 interactive dashboards to compare advanced vs emerging economies, regional growth trends, inflation outlook and country performance.
- Conducted quantitative trend analysis to identify 10+ slowdown risk economies and outperforming regions through data visualization.
- Configured slicer-driven reports from complex datasets, reducing manual analysis time by 60%.
